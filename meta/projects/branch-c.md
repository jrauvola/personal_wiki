# Branch C — Reading List

## Empirical update 2026-04-23

The F1-F6 battery on the Phase-1 CODI-at-4B anchor (see `.raw/experiments/inert_latent_hypothesis_tests`) **contests the "KV is where reasoning lives" narrative** that motivated Christopher's layer-skip / KV-steering path:

- **F4** — ablating latent-position KV entirely drops accuracy by only 25-29% relative (GSM8k 0.163→0.122, SVAMP 0.423→0.300). The format-prior template *survives* removal of the latent KV. If KV were the primary reasoning substrate we would expect accuracy to collapse toward zero, not toward a ~75%-preserved floor.
- **F5** — cross-example latent-KV swap (N=30 GSM8k pairs) leaves B's accuracy unchanged at 0.10→0.10, with only 13% of prediction texts changing. This is a **null result for KV steering at the latent-position scope on Qwen3-4B**.
- Christopher's Llama-1B KV-steering runs showed ±10-12% accuracy swings under analogous interventions.

**Reconciliation hypothesis.** Our F5 swap replays an *in-distribution* latent-KV slice from another example — one that lands in the same format-prior template basin our target example's latents also route to. Routing is preserved; output distribution is preserved. Christopher's KV perturbations are *off-manifold* — they exploit directions in KV space that Qwen3-4B's template-locked latents do not exercise. This reframes the KV-reasoning question as: *"KV reasoning matters along directions the base model is sensitive to but CODI-4B's template attractor suppresses."* The [[Quora Faithfulness Probe]] is the natural methodology extension — test whether steering along non-template PCs of the KV covariance (top PCs explain 72% of variance in the F5 proxy) recovers Christopher's effect on our 4B model.

Per spec this branch was deferred after Gate C. Status remains conditional, but the findings above re-scope its activation trigger: not "Qwen3 config bug" but rather "is the latent-KV subspace that carries reasoning the same subspace that CODI-at-4B exercises?"
