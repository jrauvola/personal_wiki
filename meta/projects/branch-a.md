# Branch A — Reading List

## Empirical update 2026-04-23

Phase 1 (V2 detach recipe on Qwen3-4B) is effectively **done as an empirical program** — it has produced the anchor results that every Phase 2 intervention will be evaluated against. The F1-F6 battery (see `.raw/experiments/inert_latent_hypothesis_tests`) fully characterizes the Phase-1 CODI-at-4B failure mode:

- **Capability collapse** — Qwen3-4B-Instruct-2507 zero-shot scores 0.86 on GSM8k; CODI V2 bf16 scores 0.16. A ~70 pp loss against the same base model.
- **Loop-lucky artifact** — V2's rising `num_latent` curve is a loop-emission routing-strength effect, not iterative computation. Track A first-sentence-trim accuracy is flat across `num_latent`; 99-100% of V2 predictions emit a looped `"The answer is: …"` substring.
- **Geometric key, not content** — F3: 7/8 latent positions decode to a fixed template `The → 0 → 0 → ? → . → . → . → .` (entropy <0.4 bits). F5: cross-example latent-KV swap leaves accuracy unchanged at 0.10→0.10 with only 13% of prediction texts changing. F6: σ=0.5 Gaussian noise on latent KV collapses accuracy to <3% — the latent sits in a narrow geometric basin whose existence routes the decoder into its format-prior attractor, but whose per-example content is irrelevant.
- **F4 floor** — ablating latent KV entirely drops accuracy by only 25-29% relative (GSM8k 0.163→0.122, SVAMP 0.423→0.300); the remaining ~70-75% of correct answers come from the same format-prior attractor the base model's zero-shot CoT reaches via genuine reasoning.

**Resolution for this branch.** Phase-1 V2 on Qwen3-4B is now the empirical anchor for Phase-2 interventions on [[CODI]]. Any Phase 2 recipe ([[SIM-CoT]], [[Latent Thoughts Tuning]], [[KaVa]]) is expected to break the F3 template lock (entropy rise on multiple positions, ideally >2 bits), flip F5 from null to non-null, shift F6 noise tolerance, and recover capability toward the 0.86 zero-shot floor. These form Branch A's Phase-2 success criteria.
